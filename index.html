Nikola Jokic's 2022-2023 Season

Background and Purpose

The purpose of this project is to showcase Nikola Jokic's incredible 2022-2023 NBA season. This interactive slideshow will show how Jokic's skillset is abnormal for a 6'11'' center.

Data

The data used for this project is from the "2022-2023 NBA Player Stats" dataset which was created by a Kaggle user using Basketball Reference. The original csv file can be found at this Kaggle link. The dataset contains common basketball stats such as points per game, minutes player per game, rebounds per game, etc.
